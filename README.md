# MultiTypeRecyclerViewAdapter
MultiTypeRecyclerViewAdapter for Recyclerview

## 特点

* 与Adapter为组合关系
* 一行代码刷新相应viewType
* 支持facebook的shimmer加载效果
* 支持粘性头
* 支持异步刷新，可扩展(如配合RxAndroid)
